# 🎵 Harmonix: AI Audio Splitter

A basic yet powerful project utilizing Meta's **HT-Demucs** model via Hugging Face. This tool uses a Hybrid Transformer architecture to separate any audio file into four clean stems.

## 🚀 How to Run
1. **Clone the repo:** `git clone https://github.com/yourusername/harmonix-audio-splitter.git`
2. **Install dependencies:** `pip install -r requirements.txt`
3. **Launch the App:** `python app.py`

## 🧠 The Tech Stack
- **Model:** `htdemucs` (Hybrid Transformer)
- **Framework:** PyTorch & Torchaudio
- **UI:** Gradio
